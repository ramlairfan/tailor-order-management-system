# tailor-order-management-system
A Tailor Order Management System built using HTML, CSS, JavaScript, PHP, and MySQL for managing customers, orders, measurements, and payments.



# Tailor Order Management System

## Features
- User Authentication
- Dashboard
- Customer Management
- Order Management
- Delivery Tracking
- Invoice Generation
- Payment Management

## Technologies
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository
2. Import `database.sql`
3. Configure database credentials in `database.php`
4. Start Apache and MySQL using XAMPP
5. Open the project in your browser
